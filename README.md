# 🚀 BancoChile Bootcamp Projects

Repositorio que contiene los proyectos desarrollados durante el **Bootcamp Full Stack Banco Chile**.

Durante este proceso trabajé diferentes tecnologías y conceptos de desarrollo de software, incluyendo programación orientada a objetos, estructuras de datos, desarrollo web frontend, consumo de APIs, manejo de lógica de negocio y buenas prácticas de programación.

---

## 🛠️ Tecnologías trabajadas

### ☕ Java
- Programación Orientada a Objetos (POO)
- Clases, interfaces y herencia
- Colecciones de Java
- Manejo de excepciones
- Algoritmos básicos
- Modelamiento de entidades
- Lógica de negocio

### 🌐 HTML / CSS / JavaScript
- Estructura semántica HTML
- Diseño con CSS
- Manipulación del DOM
- Eventos en JavaScript
- Interactividad en páginas web
