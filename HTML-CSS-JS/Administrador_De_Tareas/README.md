# 📋 TaskPro

Aplicación web de administración de tareas desarrollada con HTML, CSS y JavaScript.

## 🚀 Descripción

Este proyecto consiste en una interfaz de gestión de tareas que permite practicar la manipulación del DOM y el manejo de eventos con JavaScript.

## ✨ Funcionalidades

* Mostrar una alerta con el texto ingresado al crear una tarea.
* Marcar tareas como completadas al hacer clic sobre ellas.
* Cambiar el icono de notificaciones al pasar el cursor sobre él.

## 🛠️ Tecnologías utilizadas

* HTML5
* CSS3
* JavaScript

## 📚 Aprendizajes

Durante este proyecto se trabajó con:

* Selección de elementos del DOM.
* Event Listeners.
* Eventos `click`, `mouseover` y `mouseout`.
* Modificación dinámica de estilos y atributos.
